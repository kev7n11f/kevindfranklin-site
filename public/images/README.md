# Images

Place your images in this folder. Reference them in your config as `/images/filename.png`.

## Required Images

| Filename | Purpose | Recommended Size |
|----------|---------|------------------|
| `og-image.png` | Social media sharing preview | 1200 x 630 px |
| `agential-gold-rush-cover.png` | Book cover display | 600 x 800 px |

## Optional Project Images

Add images for your projects:
- `grievehub.png` - GrieveHub Labs
- `too-humble.png` - Too Humble Couture
- `cunningham.png` - Cunningham Business Systems
- `themebot.png` - ThemeBotPark

## Image Guidelines

1. **Format**: PNG or WebP preferred for quality
2. **Optimization**: Compress images before uploading (use TinyPNG or similar)
3. **Naming**: Use lowercase with hyphens (e.g., `my-image.png`)
4. **Aspect Ratios**:
   - Project cards: 4:3 (e.g., 800x600)
   - Book covers: 3:4 (e.g., 600x800)
   - Hero backgrounds: 16:9 (e.g., 1920x1080)

## Placeholder Behavior

If an image is not found, components will show a stylized placeholder with the first letter of the project/book title.
